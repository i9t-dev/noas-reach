Noas-reach

Noas-reach is a standalone app to manage professional relationships
Features (✅: implemented, 🚧: work in progress, 🔮: planned):
  • 🚧 Provide all the features of a plain CiviCRM standalone instance regarding contact, relationship and event management
  • 🔮 Install as a CiviCRM extension (noas-reach)
  • 🔮 Show a graph view of the relationships between contacts
  • 🔮 Find contacts via a dynamic search form
  • 🔮 Present search results as a table
  • 🔮 Perform advanced view operations: filtering, ordering, grouping
  • 🔮 Install as a standalone server-less desktop application
  • 🔮 Install as a mobile app
